GenStar · CyberShield AI Agent 🛡️

An agentic cybersecurity reasoning platform designed to detect phishing, scams, suspicious links, deepfakes, synthetic voices, and manipulated media—all powered by grounded, explainable AI reasoning.

[![Live Demo](https://img.shields.io/badge/Live_Demo-GenStar-2ea44f?style=for-the-badge&logo=vercel)](https://vigilant-whisper-guard.vercel.app/)

## 🚀 Core Features

* **Multi-Source Threat Analysis:** Analyze suspicious content across multiple input channels including **Email**, **Chat**, and **Voice**.
* **Deepfake & Media Verification:** Upload images, audio, or video files to detect synthetic manipulation, face-swapping, and voice cloning.
* **Explainable AI Reasoning:** Instead of a simple "safe/unsafe" flag, the agent provides detailed, grounded logic explaining *why* a threat was flagged.
* **Comprehensive Intelligence:** Integrated scanning for deceptive links, social engineering patterns, and malicious attachments.
